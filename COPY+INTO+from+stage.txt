*** COPY INTO from stage ***

- Command used to create the end table WEATHER:

	CREATE OR REPLACE TABLE DEMO.DEMO_SCHEMA.WEATHER (
	CITYNAME STRING,
	LAT FLOAT,
	LON FLOAT,
	CLOUDS INTEGER,
	HUMIDITY INTEGER,
	PRESSURE FLOAT,
	TEMP FLOAT,
	TIME TIMESTAMP,
	WEATHER STRING
	);


- Examples on querying straight from a stage : https://docs.snowflake.com/en/user-guide/querying-stage#query-examples


- Command used to Copy data from the weather_stage into the end table WEATHER:

	COPY INTO DEMO.DEMO_SCHEMA.WEATHER
	FROM
	(
	select
	t.$1:city:findname,
	t.$1:city:coord:lat,
	t.$1:city:coord:lon,
	t.$1:clouds:all,
	t.$1:main:humidity,
	t.$1:main:pressure,
	t.$1:main:temp,
	t.$1:time,
	t.$1:weather[0]:main
	FROM @DEMO.DEMO_SCHEMA.weather_stage t);
